# Gmail-Auto-Reply

The App allows the users to send automatic replies to non replied emails when they are on vacation

## How to run

Fork/Clone the repository to your local desktop

Run npm install to install all the dependencies

Create Credententials on Google console and paste the json file in credentials.json file

Update your email-id as tester-id in google consol panel

Run "npm run dev" to start the development server and "npm start" to start the production server
